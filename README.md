# WebService-Project

## 💻 프로젝트 개요
* HTTP, JavaScript, JDBC, JSP기반
* final_project에서 프로젝트 상세 확인

### #️⃣ 기능 설명
1. 만년달력페이지
     - 오늘 날짜 표시
     - 오늘 날짜에 맞는 년도와 월에 해당하는 달력 표시
     - 추가 구현) 오늘로 되돌아가기 버튼으로 달력 어디에서든 오늘로 돌아오기
2. 주소록 페이지
   - 이름, 전화번호, 이메일, 성별, 그룹을 입력받아 주소록 등록하기
   - 전체보기 페이지에서 3개씩 출력
   - 그룹 별 보기 페이지에서 그룹별로 주소록 출력
   - 전체보기 <-> 그룹 별 보기 페이지 이동 가능하게 구현
